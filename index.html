How to buy your own VPN access  ▼

  • Select a server.
  • Request an address by clicking on the button below the servers.
  • Make payment to the PKT address that will appear of 100 PKT.
  • Wait for a minute for the payment to be verified.
  • Request VPN credentials
  • The files will appear ready to be downloaded

How to connect on IOS  ▼

  1. Download the .mobileconfig file
  2. Go to Settings to review profile
  3. Select and install it
  4. Go to VPN to connect

How to connect on macOS  ▼

  1. Download the .mobileconfig file
  2. Click on it to install profile
  3. Go to Settings->Profiles and select it
  4. Double click to open it and click install
  5. Go to VPN to connect

How to connect on Android  ▼

  1. Download the .sswan file
  2. Install strongSwan VPN Client from Google Play, F-Droid or strongSwan download server. Launch the strongSwan VPN client.
  3. Tap the "more options" menu on top right, then tap Import VPN profile.
  4. Choose the .sswan file you transferred from the VPN server. Note: To find the .sswan file, tap the three-line menu button, then browse to the location you saved the file.
  5. On the "Import VPN profile" screen, tap IMPORT CERTIFICATE FROM VPN PROFILE, and follow the prompts.
  6. On the "Choose certificate" screen, select the new client certificate, then tap Select.
  7. Tap IMPORT.
  8. Tap the new VPN profile to connect.

How to connect on Windows  ▼

  1. Download and install openVpn client for Windows
  2. Download the .ovpn file
  3. Launch openVpn client and go to Import Profile
  4. Go to "File" and click BROWSE
  5. Select the ovpn file you downloaded
  6. Once imported simply click the switch to connect to the VPN
